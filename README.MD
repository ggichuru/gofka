# HOW TO

> for each, Open a new terminal tab

## -> Start Kafka Broker ( server ) {localhost:9092}

```bash
    make run-broker
```

## -> Run the (test) producer { notifications }

```bash
    make test-producer
```

## -> Run the (test) consumer { notifications }

```bash
    make test-consumer
```

## -> Run (simulations) notifications { bash scripts with curl }

```bash
    make simulate
```
